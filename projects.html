​We have evaluated the features with some types of CRISPR target sites for gene editing with machine learning model. We find that our machine learning model is better than the current state-of-the-art. We validated the predictive capability of this model. Our machine learning model shows improvement in comparison with previous developed model. 

TaxaCloud: An automated system with elastic cloud computing for taxonomic analytics of metagenomic amplicon data with QIIME
​The ever-increasing demand fhttp://genomes.github.io/TaxaCloud/or high-throughput metagenomic data analysis has led to an unsustainable demand for computational training and resources. Elastic computing (EC) is a service oriented computing mode that computing resources can be scaled up and down easily by the cloud service provider, such as Nectar, and Amazon Web Services (AWS). We provide a visual web application platform with a Web 2.0 user interface and an automatic scaling analysing service to support wet-lab staffs for metagenomic data analysis. This platform can request and startup virtual machine resource after that researchers submit metagenomic data for analysing by using automatic scaling analysing service. This platform has been designed to be an easy-to-use web application framework for the organisation and analysis of high-throughput metagenomic datus on bacterial 16S rRNA sequencing datasets and fungal ITS sequence datasets. It integrates QIIME (Quantitative Insights Into Microbial Ecology), a set of bioinformatics software packages and scripts, and an automatic scaling analysing service, with interactive visualisations (ownCloud web interface) to enable researchers to receive results from demultiplexing and quality filtering, OUT picking, taxonomic assignment, and diversity analysis with submitted sequencing data. These results can be used to do metagnomic analysis for microbial communities. This system has been developed and is available for use at genomes.github.io/TaxaCloud/. The related correspondent-author paper is expected to be submitted as soon as possible.

Past Projects
Genetic variation across 16 Australian bread wheat varieties
We have applied a novel algorithm SGSautoSNP for the discovery of large numbers of genomic SNPs across 16 Australian diverse bread wheat varieties. More than 10x whole genome shotgun Illumina paired read sequence data was generated through a bioplatforms collaboration and the data mapped to the draft assemblies of chromosomes 7A, 7B and 7D. This identified more than 3 million SNPs with a validated accuracy of 93%. SNP density varied along the lengths chromosome syntenic builds as well as between genomes.
SNP density analysis provides insights into the evolution and breeding history of this important crop. Furthermore, this SNP resource permits the application of high resolution skim based genotyping by sequencing, trait association and analysis of structural variation in populations. All SNPs are publically available on a GBrowse genome viewer at www.wheatgenome.info. The related first-author journal paper is Identification and characterisation of more than 4 million inter-varietal SNPs across the group 7 chromosomes of bread wheat.
WheatGenome.info
We provide a variety of web-based systems hosting wheat genome and genomic data to support wheat research and crop improvement. WheatGenome.info is an integrated database resource which includes multiple web-based applications. These in- clude a GBrowse2-based wheat genome viewer with BLAST search portal, TAGdb for searching wheat second-generation genome sequence data, wheat autoSNPdb, links to wheat genetic maps using CMap and CMap3D, and a wheat genome Wiki to allow interaction between diverse wheat genome sequencing activities. This system includes links to a variety of wheat genome resources hosted at other re- search organizations. This integrated database aims to accel- erate wheat genome research and is freely accessible via the web interface at www.wheatgenome.info. The related first-author journal paper is WheatGenome.info: An integrated database and portal for wheat genome information.
Wheat autoSNPdb
AutoSNPdb has been successfully applied to identify SNPs from Sanger sequence data for several species, including barley, rice and Brassica, but the volume of data required to accurately call SNPs in the complex genome of wheat has prevented its application to this important crop. DNA sequencing technology has been revolutionized by the introduction of next-generation sequencing, and it is now possible to generate several million sequence reads in a timely and cost-effective manner. We have produced wheat transcriptome sequence data using 454 sequencing technology and applied this for SNP discovery using a modified autoSNPdb method, which integrates SNP and gene annotation information with a graphical viewer. A total of 4 694 141 sequence reads from three bread wheat varieties were assembled to identify a total of 38 928 candidate SNPs. Each SNP is within an assembly complete with annotation, enabling the selection of polymorphism within genes of interest. It is available online at here. The online demonstration has been published on YouTube. The related first-author journal paper is Single Nucleotide Polymorphism Discovery from Wheat Next Generation Sequence Data.
